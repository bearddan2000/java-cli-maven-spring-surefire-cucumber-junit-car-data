# java-cli-maven-spring-surefire-cucumber-junit-car-data

## Description
A POC for spring app using JUnit5
and cucumber framework with surefire plugin.
Define a parameter type of `example.model.Car`
in the test runner class.

## Tech stack
- java
- maven
  - spring
  - junit
  - surefire
  - cucumber

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
